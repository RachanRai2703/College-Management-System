## 🆘 QUICK TROUBLESHOOTING

| Issue | Solution |
|-------|----------|
| Port already in use | Run `start-all.bat` from fresh terminal |
| npm packages missing | Run `npm install` in both folders |
| Demo login fails | Run `npm run seed` in backend |
| Frontend can't reach backend | Check `.env` has correct API URL |
| Attendance not showing | Ensure backend seeded with `npm run seed` |
